# Cybersecurity Incident Reporting and Tracking System

## Project Description

A web-based platform for reporting, tracking, and managing cybersecurity incidents.

## Features

- User Registration and Login
- Incident Reporting
- Incident Status Tracking
- Dashboard Statistics
- Evidence Management
- Activity Logs

## Technology Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- FastAPI

### Database
- PostgreSQL (Supabase)

### Deployment
- Vercel
- Render
